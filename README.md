# clj-pam

clj-pam is a Clojure wrapper for the JPam library, providing Linux PAM authentication for Clojure apps.

## Usage

The most basic usage is simply:

```clojure

(use 'clj-pam.auth)

(simple-auth "username" "password")
```

More examples to follow.
